# 📊 Administrador de Finanzas

Una aplicación web simple para gestionar gastos y deudas personales.

## 🌟 Características

- ✅ Gestión de personas
- ✅ Registro de gastos y deudas
- ✅ Gastos recurrentes automáticos
- ✅ Filtros por persona y fecha
- ✅ Almacenamiento local (IndexedDB)
- ✅ Diseño responsive
- ✅ Interfaz moderna con Tailwind CSS

## 🚀 Despliegue Online

### Opción 1: GitHub Pages (Gratis)

1. Crea un repositorio en GitHub
2. Sube todos los archivos
3. Ve a Settings > Pages
4. Selecciona "Deploy from a branch"
5. Elige la rama `main` y carpeta `/ (root)`
6. Tu app estará disponible en: `https://tuusuario.github.io/turepositorio`

### Opción 2: Netlify (Gratis)

1. Ve a [netlify.com](https://netlify.com)
2. Arrastra la carpeta del proyecto
3. ¡Listo! Tu app estará online

### Opción 3: Vercel (Gratis)

1. Ve a [vercel.com](https://vercel.com)
2. Conecta tu repositorio de GitHub
3. ¡Despliegue automático!

## 📱 Uso

1. **Agregar Personas:** Ve a la sección "Personas" y agrega los nombres
2. **Registrar Gastos:** Ve a "Agregar Gasto" y completa los datos
3. **Ver Resumen:** En "Panel Principal" puedes filtrar y ver todos los gastos
4. **Gastos Recurrentes:** Marca la casilla para gastos mensuales automáticos

## 💾 Datos

- Los datos se guardan localmente en tu navegador
- No se envían a ningún servidor
- Cada dispositivo tiene su propia base de datos

## 🛠️ Tecnologías

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript (Vanilla)
- IndexedDB para almacenamiento
- Font Awesome para iconos
